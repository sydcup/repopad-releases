This repository hosts the public release artifacts for **RepoPad**. Builds here are produced automatically by the CI pipeline whenever a new version is tagged — every release ships a notarized `RepoPad-<version>.zip` containing `RepoPad.app`.

## Install

### Homebrew

```sh
brew install --cask sydcup/repopad/repopad
```

### Direct download

1. Grab the latest `RepoPad-<version>.zip` from the [Releases](https://github.com/sydcup/repopad-releases/releases) page.
2. Unzip it.
3. Drag `RepoPad.app` into your `Applications` folder.

## Requirements

- macOS Tahoe (15) or later
- Apple silicon or Intel
